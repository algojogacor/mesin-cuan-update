# 🎯 Workflow: Boost Script & Prompt Enhancer
**Objective:** Optimize the LLM prompts in Mesin Cuan for maximum viewer retention and viral potential.

**Instructions for Agent:**
1. **Activate Skill:** Call upon the `@content-strategy` skills from the repository.
2. **File Inspection:** Open and read the prompt file.
3. **Analyze & Refactor:** Evaluate any prompt variable. You must ensure it strictly enforces the "Retention-Focused Horror" framework:
   - **The 3-Second Hook:** The AI must generate scripts that start with a high-impact, curiosity-inducing opening statement.
   - **Aggressive Pacing:** Force the LLM to use short, punchy sentences (this is mandatory to sync perfectly with our `\an2` Split-Word subtitle engine).
   - **Visual Directives:** Require the LLM to include clear, bracketed visual cues (e.g., `[Show an abandoned hospital]`) to guide the `footage_engine`.
4. **Execution:** If the current prompt is too generic, rewrite and update `SYSTEM_PROMPT_SHORTS` directly in the file to make it more aggressive, dark, and retention-optimized.
5. **Report:** Output a brief summary of the changes made and confirm the prompt is ready for the next batch render.
